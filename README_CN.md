## 简介

Oh My GitHub 是一款现代、简洁的 GitHub 桌面应用。所有入口都汇聚在侧边栏，让你在
通知、Pull Request、代码评审、Issue 与 Actions 之间无缝切换——无需反复刷新页面，
也不必堆积浏览器标签。

## 功能亮点

- **简洁强大的预览页** — 所有功能都在侧边栏一触即达：无需新建标签、无需刷新页面，更不用等待加载，打开即可纵览仓库全貌。
- **丝滑的列表管理** — Pull Requests、Issues 和 Actions 一览无余，筛选、打开、操作一气呵成，每一次交互都干脆利落。
- **右侧面板即时预览** — 无需离开当前页面，即可在右侧面板预览各类文件：代码、Diff 都能直接渲染，PR 的每一处改动一眼看清。
- **实时的 Actions 详情** — 工作流的运行进度实时呈现，每个 Job 的状态与完整日志同步滚动，无需反复刷新。
- **侧边栏书签** — 任何页面都能收藏进侧边栏书签，常用的仓库、PR 和视图触手可及。
- **个性化定制** — 多种主题与样式任你挑选，还支持自定义界面字体与代码块样式，把界面调成你最顺眼的样子。
- **多账号快速切换** — 支持登录多个 GitHub 账号，一键即可切换，工作与个人身份无缝转换。
